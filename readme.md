# First Project Java Spring

A Spring Boot MVC web application built with Java, demonstrating basic Spring controllers, Thymeleaf templating, and HTTP request handling.

## Technologies Used
- Java 17
- Spring Boot 3.0
- Spring MVC
- Thymeleaf
- Lombok
- Maven

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run `FirstProjectJavaSpringApplication.java`
4. Open your browser and go to `localhost:8080`
